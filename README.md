# Mobile-Sales
🚀 Power BI Project: Mobile Sales Dashboard 📊

I’m excited to share my latest Power BI project where I built an interactive Mobile Sales Dashboard to analyze sales performance and customer trends.

🔧 Data Preparation & Cleaning

Combined Day, Month, and Year into a single Date column using a custom column
Converted data types appropriately for better analysis
Standardized values (e.g., converted Sun → Sunday  and cleaned inconsistent entries)
Ensured data accuracy and consistency for reliable insights

📊 Key KPIs (Cards)

💰 Total Sales: 3M+
📦 Total Quantity: 83
 🧾 Total Transactions: 15
 📈 Average Sales: 35K

📈 Dashboard Highlights

Sales Trend Analysis: Line chart showing total sales by day
Product Insights: Sales comparison by mobile models
Geographical View: Sales distribution across cities (map visualization)
Customer Insights: Ratings analysis to understand customer satisfaction
Payment Analysis: Transaction breakdown by payment methods
Dynamic Filters: Month-wise slicer, brand selection, and payment method filters for better interactivity
